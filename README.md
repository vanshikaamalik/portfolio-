# Portfolio Website
A responsive multi-page portfolio website built using HTML, CSS, and JavaScript.
It includes sections for profile, skills, projects, achievements, and contact information.
## Features
- Welcome landing page with navigation buttons
- Profile, Skills, Projects, Achievements, and Contact sections
- Smooth transitions and animations
- Responsive design for desktop and mobile
## Technologies Used
- HTML
- CSS
- JavaScript
## How to Run
- Clone the repository
- Open index.html in any browser
